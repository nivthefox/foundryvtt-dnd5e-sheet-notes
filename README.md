# 5e Sheet Notes & Trackers

A Foundry VTT module that adds a comprehensive Notes tab to D&D 5th Edition character sheets, featuring integrated note-taking and custom tracker functionality.

## Features

- **Notes Tab**: Adds a new "Notes" tab to both PC and NPC character sheets
- **Journal Integration**: Link and organize JournalEntry documents with your characters
- **Custom Trackers**: Create string, number, and enumeration trackers for any character data
- **Drag & Drop**: Seamlessly drag journal entries between the Notes tab and Journal sidebar
- **Categories**: Organize notes and trackers with customizable categories

## Installation

1. In Foundry VTT, go to the Add-on Modules tab
2. Click "Install Module"
3. Search for "5e Sheet Notes & Trackers" (coming soon to official package repository)
4. Click "Install"

### Manual Installation

1. Download the latest release
2. Extract the files to your Foundry VTT modules folder
3. The folder should be named `foundryvtt-5e-sheet-notes`
